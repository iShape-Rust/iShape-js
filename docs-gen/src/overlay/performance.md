All tests were run on a machine with the following specifications:
**3 GHz 6-Core Intel Core i5, 40GB 2667 MHz DDR4**

- **Rust-iOverlay** v1.5.2
- **Swift-iOverlay**  v1.13.0
- **Clipper2**  v1.4.0
